    *进入店家入驻页面点击我要入驻
    * 确认入驻协议、查看入驻须知、录入合作门店联系人信息

    *确认在线服务协议
3.等待审核
    *7个工作日返回审核结果
    *店家可以查询审核进度及接收入驻进度邮件
4.门店缴费、开店
    * 店家在线缴费
    * 确认缴费无误
    *开通门店
    *店家登陆后台
